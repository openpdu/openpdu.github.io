# OpenPDU HCL (Hardware Compatibility List)

## controller board
- zero (H2 + 256/512Mb + 26pin)
- oneplus (H6 + 1Gb + 26pin)
- one (H3 + 512Mb + 40pin)
- pc (H3 + 1Gb + 40pin)
- pcplus (H3 + 1Gb + 40pin)
- pc2e (H3 + 2Gb + 40pin)
- pc2 (H5 + 1Gb + 40pin)
- plus2 (H3 + 2Gb + 40pin)
- Win (A64 + 1Gb + 40pin)
- WinPlus (A64 + 2Gb + 40pin)
- Prime (H5 + 2Gb + 40pin)
- zeroplus (H5 + 512MB + 26pin)
- R1 (H2 + 256Mb + 26pin) - dual ETH
